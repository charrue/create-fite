
create:page
用于快速创建小程序页面
- 选择是否创建tabbar页面
  - 如果是一级页面，则在填写当前页面所属的一级页面`parent`
- 填写页面标题`title`和页面名`name`
- 创建以下文件
  - `pages/tabbar|{parent}/{name}`
  - `pages/tabbar|{parent}/{name}.js`
  - `pages/tabbar|{parent}/{name}.json`
  - `pages/tabbar|{parent}/{name}.wxml`
  - `pages/tabbar|{parent}/{name}.less`
